DebugInboundEventsHandler

public class DebugInboundEventsHandler : ChannelInboundHandler

ChannelInboundHandler that prints all inbound events that pass through the pipeline by default, overridable by providing your own closure for custom logging. See DebugOutboundEventsHandler for outbound events.
